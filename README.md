# Smart City 360 Assistant

An AI-powered Smart City Assistant built using Streamlit and Hugging Face Transformers. This application simulates intelligent urban management with multiple AI-based features.

---

## Overview

This project demonstrates how artificial intelligence can be used in smart city systems for better decision-making, citizen interaction, and sustainability.

---

## Features

1. City Health Dashboard
   Displays important city metrics like air quality, traffic conditions, and energy usage.

2. Citizen Feedback System
   Allows users to report issues. The system summarizes the complaint and routes it to the appropriate department.

3. Document Summarization
   Users can upload PDF or text files and get a summarized version using NLP models.

4. Eco Advice Assistant
   Provides suggestions for eco-friendly practices using AI.

## How to Run the Project

Step 1: Install required libraries
pip install -r requirements.txt

Step 2: Run the application
streamlit run main_app.py
